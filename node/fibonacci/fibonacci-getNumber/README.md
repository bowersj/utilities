# Generating One or More Fibonacci Numbers

### Abstract

This is a quick attempt at building a function that returns the fibonacci number at the provided position.


### Examples
```javascript
fib.upTo( 1 );  // 1
fib.upTo( 2 );  // 1
fib.upTo( 3 );  // 2
fib.upTo( 4 );  // 3
fib.upTo( 5 );  // 5
fib.upTo( 11 ); // 89
```
